# Devcon Language Server

This package provides a language feature for the DevCon language for visual
studio code.

# Quickstart

Run `yarn deps; yarn watch` and then start VS code using the VS Code launch
option in your debug panel.

# Scripts

| script  | description                                                    |
| ------- | -------------------------------------------------------------- |
| deps    | Build CSharp dependencies and install them in node_modules     |
| compile | Build the project typescript and write it to the out directory |
| watch   | Watch typescript files and compile when they change            |
